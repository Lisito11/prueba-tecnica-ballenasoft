## Correr el proyecto

**Node v16.17.18**

Para instalar las dependencias
```bash
yarn 
```
Para correr el proyecto
```bash
yarn dev
```

En caso de que sea necesario cambiar la url de la api en 

```bash
src/api/api.ts

const development = "URL_API"
```
